Variable - это именованная область памяти,
содердимое которой может изменяться в процессе выполнения программы.

//Синтаксис объявления переменной:
			type name;
type - тип объявляемой переменной.
Тип определяет три вещи:
	1. Сколько памяти переменная будет занимать;
	2. Какие значения она сможет принимать;
	3. Какие операции над ней можно выполнять;

			Типы данных:
1. Логические типы: bool (Boolean) - 1 Byte, true false.
	true  - 1;
	false - 0;
2. Символьные типы: char (Character - Символ) 1 Byte
	ASCII - American Standard Code for Information Interchange
	256
	1 Byte = 8 bit;
	bit - Binary Digit (Двоичная цифра 0 или 1) => Байт - это восьмиразрядное двоичное число.
	Переменная типа 'char' может хранить 1 из 256 ASCII-символов.
3. Числовые типы: целочисленные, вещественные
	unsigned - беззнаковые	(могут хранить только положительные целые числа);
	  signed - знаковые		(могут хранить как положительные, так и отрицательные целые числа);

	  short, long, int, long long;
