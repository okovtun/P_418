Git != GitHub
Git - это система контроля версий
GitHub - это облачное хранилище

Repository - это хранилище исходных кодов.
fetch - скачивает изменения с удаленного репозитория на локальный;
pull  - скачивает и распаковывает изменения с уделенного репозитория на локальный;
push  - отправляет изменения с локального репозитория на удаленный;

COMMIT - это контрольная точка, которая содржит снимок репозитория в какой-то момент времени;
