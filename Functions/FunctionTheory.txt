type name(parameters)
{
	....;
	group-of-statements;
	....;
}

type - тип возвразаемого значения.
void - пустота.

return value;
return;

name - имя функции.

parameters - принимаемые параметры функции.

group-of-statements - это тело функции.
В теле функции можно писать все что угодно,
кроме РЕАЛИЗАЦИИ другой функции;
